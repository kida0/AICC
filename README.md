# AICC(AI Call Center) 토이 프로젝트

## AICC를 만들어보는 이유

자사에서 AICC를 도입해 운영하면서 기능적인 아쉬움이 있었는데요.

- 주변 소음이 심한 환경에서 STT 인식률이 눈에 띄게 떨어진다
- 인식 오류가 누적되면 대화 흐름 전체가 어긋난다

STT 앞단에 노이즈 제거를 붙여보면 어떨까 하는 관심이 생겼고, 이를 직접 실험해보기 위해 Claude Code로 AICC 시스템을 구성해보았습니다.



## AICC의 구성

AICC는 자동 전화 발신과 실시간 AI 음성 대화를 목표로 하며, 다음과 같이 구성되어 있습니다.

- **전화 발신**: 전화번호 입력 시 Twilio를 통해 자동 발신
- **실시간 대화 파이프라인**: STT (Whisper) → LLM (Claude) → TTS (OpenAI)
- **녹음 및 저장**: 모든 통화를 자동으로 녹음하고 보관
- **실시간 Transcript**: 통화 중 대화 내용을 텍스트로 표시
- **통화 내역 관리**: 과거 통화 조회, 재생, transcript 확인

**기술 스택**: FastAPI, PostgreSQL, Redis, React, Twilio, OpenAI Whisper/TTS, Claude Code
